# Project DNA

## Purpose

This file captures the enduring design, business, and working identity of the Service Performance Report module.

## Report Purpose

Service performance reporting for operational visibility and management decision support.

## Data Platform

- SAP HANA via ODBC
- Company schema: CANON
- Currency: IQD

## Design Language

- Follows portfolio-wide visual identity standard.
- Theme tokens from `Shared/Standards/portfolio-theme.tokens.json`.

## Working Method

- PBIP is the editable source of truth.
- Review from packaged artifacts, not raw PBIP folders.
- Screenshot capture and memory update after meaningful edits.

Do not use this file as a change log.
