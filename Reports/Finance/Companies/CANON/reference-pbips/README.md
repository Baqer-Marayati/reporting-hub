# Reference PBIP uploads

Drop full **PBIP folders** here (the folder that contains `*.pbip`, `*.Report/`, and `*.SemanticModel/`).

After you copy them in, tell the agent the folder name(s) so they can read `definition/pages/**/visuals/**/visual.json` and other report definition files.

Do not commit sensitive or licensed third-party reports if your policy forbids it.
