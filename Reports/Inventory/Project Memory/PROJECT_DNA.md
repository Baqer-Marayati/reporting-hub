# Project DNA

## Purpose

This file should capture the enduring design, business, and working identity of the Inventory Performance Report module.

Use it for:
- report purpose
- intended audience
- enduring design language
- core working method

Do not use it as a change log.

## Enduring Visual Identity Rule

- Inventory must follow the portfolio visual identity baseline from Finance:
  - `Shared/Standards/portfolio-visual-identity.md`
  - `Shared/Standards/portfolio-theme.tokens.json`
- This is a default constraint, not a one-time preference.
