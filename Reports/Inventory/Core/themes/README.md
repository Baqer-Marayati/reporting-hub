# Themes

This folder stores reusable Inventory theme files.

## Current Baseline

- `Inventory.PortfolioTheme.json`
  - Generated from `Shared/Standards/portfolio-theme.tokens.json`
  - Intended to keep Inventory aligned with the portfolio visual identity derived from Finance

## Usage

In Power BI Desktop:
1. Open the Inventory PBIP file.
2. Go to `View` -> `Themes` -> `Browse for themes`.
3. Select `Reports/Inventory/Core/themes/Inventory.PortfolioTheme.json`.
