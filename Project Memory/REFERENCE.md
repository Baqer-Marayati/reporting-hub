## Benchmark Direction
From `Design Benchmarks/Sample 2`:
- centered white report canvas with a strong framed report feel
- clean header band / page-shell system
- disciplined card row across the top
- slicers stacked on the left rail
- airy page spacing with large white visual containers
- conservative finance palette with strong blue emphasis
- professional CFO / management-report tone

## Visual Principles
- Keep spacing disciplined.
- Avoid crowded pages.
- Preserve a professional CFO / management-report tone.
- Keep layout hierarchy clear: filters, KPIs, analysis, detail.
- Prefer benchmark accuracy over style experimentation.
- Prefer starting from a benchmark shell over trying to force the old shell to match.

## Records Maintenance
- Re-check `Design Benchmarks` before major layout, color, or style decisions.
- Prefer newer screenshots and templates when they clearly reflect the latest taste.
- When the benchmark changes, refresh this file so it reflects the latest visual direction.
